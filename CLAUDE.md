# harshrathore.me — personal site

Terminal-style single-page portfolio for Harsh Rathore, served by GitHub Pages
at https://harshrathore.me (repo: `HarshRat/HarshRat.github.io`, branch: `master`).

## Architecture

- **`index.html`** — the entire site: one file with inline CSS/JS, no build step,
  no framework. Terminal aesthetic (JetBrains Mono, dark-only, green accent).
  Content is organised as `.block` sections, each styled as a shell command
  (`whoami`, `cat about.txt`, `history --work`, `ls ~/projects`, `travel --map`,
  `spotify --status`, `contact --list`).
- **Boot animation** — on load, each block's command is typed out then its output
  revealed (`data-cmd` attribute drives it). Skipped when the URL has a `#hash`
  or `prefers-reduced-motion` is set — anchors must keep working.
- **Interactive prompt** — the input at the bottom is a real mini-CLI (`help`,
  `about`, `work`, `projects`, `travel`, `music`, `contact`, `sudo hire-me`,
  jokes). New sections should get a command + a `HELP` entry.
- **`assets/map-asia.svg`, `assets/map-india.svg`** — generated dot-matrix travel
  maps. **Do not hand-edit**; regenerate with the script (see below).
- **`assets/travel/*.jpg`** — travel photos, compressed to ≤ ~250 KB each
  (`sips -s formatOptions 55 -Z 1000`). Lightbox captions live in `data-cap`
  attributes in `index.html`.
- **`worker/`** — Cloudflare Worker (`harshrathore-music`) that proxies the
  Spotify API for the music section. Deployed separately from the site.

## Deploying

- **Site**: push to `master` → GitHub Pages redeploys in ~1 min. No CI.
- **Worker**: `cd worker && npx wrangler deploy` (needs `wrangler login`).
- Custom domain via `CNAME` file (`harshrathore.me`) — do not delete it.
- DNS at Namecheap: 4 × A records `@` → GitHub Pages IPs (185.199.108–111.153),
  `www` CNAME → `harshrat.github.io`. The same zone also carries email
  forwarding (MX/SPF for mail@harshrathore.me) and a `meeting.with` URL-redirect
  to Calendly — never remove those.

## Music section (Spotify)

- Frontend fetches `https://harshrathore-music.harsh-15c.workers.dev`:
  `GET /now` (now playing / last played, edge-cached 30 s, polled every 45 s)
  and `GET /top` (top artists/tracks, 4-week + 6-month windows, cached 1 h).
- Worker secrets (set via `npx wrangler secret put`, values in Cloudflare only,
  NEVER in this repo): `SPOTIFY_CLIENT_ID`, `SPOTIFY_CLIENT_SECRET`,
  `SPOTIFY_REFRESH_TOKEN`.
- Spotify app: "harshrathore.me music" on Harsh's Spotify developer dashboard,
  client ID `923ea36464214f1e9141330ca767ada4`, redirect URI
  `http://127.0.0.1:8888/callback`, scopes `user-top-read
  user-read-currently-playing user-read-recently-played`.
- ⚠️ The app is in **development mode**: refresh tokens live ~180 days. Around
  **Jan 2027** the section will start failing (it degrades silently — the
  "connecting to spotify…" line times out and top lists stay hidden). Fix: rerun
  the one-time OAuth (local catcher on 127.0.0.1:8888 → exchange code at
  accounts.spotify.com/api/token → `wrangler secret put SPOTIFY_REFRESH_TOKEN`).
- CORS allowlist in `worker/src/index.js` covers harshrathore.me, www, and
  localhost:8420 — extend it if the preview port changes.

## Travel maps

Generated by a Python script (session scratchpad, `genmaps.py` — recreate from
this description if lost): rasterises GeoJSON land polygons onto a dot grid via
point-in-polygon, emits compact SVG paths.

- Asia map: Natural Earth 110m countries, **except India, which uses official
  Indian boundaries** (full J&K + Ladakh incl. PoK and Aksai Chin) from the
  district-level dataset at `udit-001/india-maps-data` — this was an explicit
  request; keep it when regenerating.
- India map: same district dataset, visited states lit green. Visited: Kerala,
  Tamil Nadu, Karnataka, Andhra Pradesh, Goa, Maharashtra, MP, Rajasthan,
  Gujarat, Odisha, Himachal, Uttarakhand, Ladakh, Delhi, Telangana (15).
- Markers: Beas Kund trek + "indore · home" (Indore, not Bengaluru) on the India
  map; "ijen · bromo" volcano marker + explicit Singapore dot (too small for
  110m data) on the Asia map.
